# 📚 Student Task Manager

A simple and clean **task management web app** designed for students to organize subjects, assignments, and deadlines efficiently.

---

## 🚀 Features

* Add tasks with:

  * Subject 📘
  * Task description 📝
  * Deadline 📅
  * Priority (Low / Medium / High) ⚡
* Mark tasks as completed ✅
* Delete tasks ❌
* Data stored in browser (LocalStorage) 💾
* Clean and minimal UI 🌙

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* LocalStorage (for persistence)
